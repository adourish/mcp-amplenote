# Privacy Policy

**mcp-amplenote** is a Claude Code plugin that connects to your personal Amplenote account.

## Data Collection

This plugin does **not** collect, transmit, or store any personal data. All operations run locally on your machine.

## Data Access

The plugin accesses your Amplenote account solely to execute the actions you request through Claude (creating, reading, updating, searching, and deleting notes). No data is sent to any third party other than the Amplenote API at `api.amplenote.com`.

## Credentials

Your Amplenote OAuth credentials are stored locally in a config file that you control. The plugin reads and updates this file only to maintain a valid access token. Credentials are never logged or transmitted beyond the Amplenote API.

## Contact

For questions or concerns, open an issue at https://github.com/adourish/mcp-amplenote/issues
